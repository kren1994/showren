# 連珠 検討用ボード

ブラウザで動作する連珠の棋譜検討ツールです。`index.html` を開くだけで使えます。

---

## 操作方法

### 石を置く / 変化手を作る

- 盤面をクリック（またはタップ）すると石を置きます。
- 同じ局面から別の交点をクリックすると **分岐（変化手）** が作られます。
- すでに存在する子ノードの交点をクリックすると、その手順に **移動** します。

### 局面の移動

| 操作 | キーボード | ボタン |
|---|---|---|
| 1手戻る | `←` / `j` | `<` |
| 1手進む | `→` / `k` | `>` |
| 前の分岐点へ | `↑` / `h` | `<<` |
| 次の分岐点へ | `↓` / `l` | `>>` |
| 右クリック | — | 1手戻る |

### 編集

| 操作 | 説明 |
|---|---|
| `Delete` | 現在の手（以降の変化含む）を確認後に削除。関連する索引ラベルも削除 |
| `Ctrl+クリック` | 交点にテキストラベルを付ける |

### ファイル

| 操作 | 説明 |
|---|---|
| `Ctrl+S` / ファイルアイコン → `保存` | 保存（ファイルを開いている場合は上書き） |
| `Ctrl+O` / ファイルアイコン → `読込` | JSONファイルを開く |
| ドラッグ＆ドロップ | JSONファイルを盤面にドロップして読み込む |

> ブラウザが File System Access API に対応していない場合（iOS Safari 等）は `Ctrl+S` でダウンロード保存になります。

### 表示切替

| 操作 | 説明 |
|---|---|
| 目アイコン → `コメント` / `Ctrl+F` | ヒント（次の手の丸印・ラベル・コメント欄）の表示切替 |
| 目アイコン → `座標` | 座標ラベルの表示切替 |

### コメント（PC のみ）

| 操作 | 説明 |
|---|---|
| `Ctrl+;` | コメントタブ時はコメント入力欄と盤面の切替 |
| `Ctrl+Shift+;` | コメント / 索引 タブ切替 |
| `Ctrl+J` | コメント欄内で改行挿入 |
| `Ctrl+H` | コメント欄内で1文字削除 |

> タッチメイン端末（スマホ・タッチ専用タブレット）ではコメント入力は無効です。

### 索引

- 右ペインの `索引` タブから、現在局面に 3 階層ラベルを登録できます。
- ラベル一覧は `第1階層` 降順、`第2/第3階層` 自然順で表示されます。
- 一覧の葉行をクリックすると代表手順へ移動できます。代表手順が消えている場合は同一局面の探索にフォールバックします。
- 一覧の葉行から移動した場合、その局面へ向かう現在の代表経路も更新されるため、戻ってから `>` で進むと同じ手順をたどれます。
- ラベルは 1 局面につき 1 つ、かつラベル名も重複不可です。
- ラベル削除は一覧の葉ノード単位で行います。
- 登録後も入力欄の内容は保持されるため、連続登録がしやすくなっています。
- 索引タブで `Ctrl+;` を押すと、第1階層入力欄と盤面を切り替えます。
- タッチメイン端末では索引の登録欄は非表示で、一覧表示とジャンプのみ使えます。

## 注意事項

- ページを閉じる前に確認ダイアログが表示されます（誤操作防止）。
- 右下のフローティングタブで `コメント` と `索引` を切り替えられます。
- スマホ・タブレット幅では操作ボタンが画面下部のバーに移動します。ファイル操作はファイルアイコン、表示切替は目アイコンから行います。
