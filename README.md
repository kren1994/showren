# 連珠 検討用ボード

ブラウザで動作する連珠の棋譜検討ツールです。`index.html` を開くだけで使えます。

---

## 操作方法

### 石を置く / 変化手を作る

- 盤面をクリック（またはタップ）すると石を置きます。
- 同じ局面から別の交点をクリックすると **分岐（変化手）** が作られます。
- すでに存在する子ノードの交点をクリックすると、その手順に **移動** します。

### 局面の移動

| 操作 | キーボード | ボタン |
|---|---|---|
| 1手戻る | `←` / `j` | `<` |
| 1手進む | `→` / `k` | `>` |
| 前の分岐点へ | `↑` / `h` | `<<` |
| 次の分岐点へ | `↓` / `l` | `>>` |
| 右クリック | — | 1手戻る |

### 編集

| 操作 | 説明 |
|---|---|
| `Delete` | 現在の手（以降の変化含む）を削除 |
| `Ctrl+Z` | Undo（最大50手） |
| `Ctrl+クリック` | 交点にテキストラベルを付ける |

### ファイル

| 操作 | 説明 |
|---|---|
| `Ctrl+S` | 保存（ファイルを開いている場合は上書き） |
| `Ctrl+O` / 読込ボタン | JSONファイルを開く |
| ドラッグ＆ドロップ | JSONファイルを盤面にドロップして読み込む |

> ブラウザが File System Access API に対応していない場合（iOS Safari 等）は `Ctrl+S` でダウンロード保存になります。

### 表示切替

| 操作 | 説明 |
|---|---|
| `A` ボタン / `Ctrl+F` | ヒント（次の手の丸印・ラベル・コメント欄）の表示切替 |
| `座標` ボタン | 座標ラベルの表示切替 |

### コメント（PC のみ）

| 操作 | 説明 |
|---|---|
| `Ctrl+;` | コメント入力欄にフォーカス / フォーカス解除 |
| `Ctrl+J` | コメント欄内で改行挿入 |
| `Ctrl+H` | コメント欄内で1文字削除 |

> タッチメイン端末（スマホ・タッチ専用タブレット）ではコメント入力は無効です。

## JSONフォーマット

保存ファイルは UTF-8 の JSON です。トップレベルは以下の4フィールドです。

```json
{
  "t": { ... },
  "p": { ... },
  "c": "現在のノードID",
  "n": 次のノード番号
}
```

| フィールド | 説明 |
|---|---|
| `t` | 棋譜ツリー（ノードIDをキーとするオブジェクト） |
| `p` | 局面データベース（局面キーをキーとするオブジェクト） |
| `c` | 現在表示中のノードID |
| `n` | 次に採番するノード番号 |

---

### ツリーノード (`t` の値)

各ノードは1手を表します。ルートノードのIDは `"r"` です。

```json
"r": {
  "i": "r",
  "p": null,
  "c": ["1", "2"],
  "x": null,
  "y": null,
  "o": 0,
  "m": 0,
  "l": "1"
}
```

| フィールド | 説明 |
|---|---|
| `i` | このノード自身のID（ルートは `"r"`、それ以外は採番された整数文字列） |
| `p` | 親ノードのID（ルートは `null`） |
| `c` | 子ノードIDの配列 |
| `x` | 交点の列（0〜14、ルートは `null`） |
| `y` | 交点の行（0〜14、ルートは `null`） |
| `o` | 石の色（`1`=黒、`2`=白、`0`=ルート） |
| `m` | 手数（ルートは `0`） |
| `l` | 最後に選択した子ノードのID（省略可） |

---

### 局面データベース (`p` の値)

局面の対称性を考慮したカノニカルキーをキーとします。

```json
"ABcd...xyz1": {
  "c": "ここに検討コメント",
  "l": { "7,7": "A", "8,7": "B" },
  "n": { "7,7": 1, "8,7": 1 }
}
```

**キーの形式：** 盤面225マス（15×15）の石配置を 5トリット/バイト で45バイトに圧縮し、base64url エンコードした60文字＋手番色1文字の計61文字。

| フィールド | 説明 |
|---|---|
| `c` | 局面コメント（文字列） |
| `l` | 交点ラベル。キーは `"x,y"` 形式（カノニカル座標）、値は表示テキスト |
| `n` | 次の手の登録済み交点。キーは `"x,y"` 形式、値は `1` |

> `l` と `n` の座標はカノニカル座標（対称変換後）で記録されます。盤面上の物理座標とは向きが異なる場合があります。

---

## 注意事項

- ページを閉じる前に確認ダイアログが表示されます（誤操作防止）。
- Undo はメモリ上のみで、ファイルには記録されません。
- 旧フォーマット（フィールド名が短縮前）の JSON も読み込み可能です。
