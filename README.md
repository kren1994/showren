# 連珠 検討用ボード

ブラウザで動く、連珠の検討用ボードです。GitHub Pages でそのまま使えます。

## できること

- 盤面をクリックまたはタップして着手できます。
- 同じ局面から別の場所に打つと、分岐を作れます。
- すでに表示されている候補手をクリックすると、その分岐へ進みます。
- 盤上の既存の石をクリックすると、`>` ボタンと同じように 1 手進みます。
- 交点に短い文字や記号を置いて、検討メモを残せます。
- 局面ごとのコメント、候補手、索引ラベルをまとめて保存できます。

## 使い方

1. [公開ページ](https://kren1994.github.io/showren/) をブラウザで開きます。
2. 盤面をクリックして手順を入力します。
3. 必要に応じてコメントや交点テキスト、索引ラベルを追加します。
4. ファイルメニューの `読込`、`保存`、`書出` を使ってデータを扱います。

## 基本操作

### 着手と編集

- 空いている交点をクリックまたはタップすると着手します。
- 現在局面から別の交点に打つと、新しい分岐を作ります。
- 候補手の石をクリックすると、その候補手へ移動します。
- 盤上の既存石をクリックすると、主分岐の次の 1 手へ進みます。
- `Ctrl+クリック` で交点に文字や記号を付けられます。

### 手順の移動

| 操作 | キーボード | ボタン |
|---|---|---|
| 1手戻る | `←` / `j` | `<` |
| 1手進む | `→` / `k` | `>` |
| 前の分岐点へ | `↑` / `h` | `<<` |
| 次の分岐点へ | `↓` / `l` | `>>` |
| 現在手以降を削除 | `Delete` | なし |

- 盤面上で右クリックしても 1 手戻れます。

## ファイルの読込・保存・書出

- ファイルメニューの `読込` または `Ctrl+O` で JSON ファイルを開けます。
- ファイルメニューの `保存` または `Ctrl+Shift+S` で現在の内容と設定項目をブラウザ内に保存できます。
- ファイルメニューの `書出` または `Ctrl+S` で JSON ファイルを書き出せます。
- `保存` はブラウザ内保存、`書出` は JSON ファイルの出力です。
- JSON ファイルを画面へドラッグ＆ドロップして読み込むこともできます。

## ファイルメニュー

- `言語` で日本語と中国語を切り替えられます。
- `一手削除` で現在の手以降の変化を削除できます。

## 表示メニュー

目アイコンのメニューから、盤面表示を切り替えられます。

- `座標`: 座標表示のオン・オフ
- `盤面色`: `茶` → `緑` → `灰` → `白` の順で切替
- `タブ`: 右側の `コメント` / `索引` タブ表示のオン・オフ
- `手数`: 石の中の手数表示のオン・オフ
- `記号等`: 候補手の点、交点テキスト、コメント入力の表示切替

キーボード操作:

- `Ctrl+F`: `記号等` の表示切替
- `Ctrl+Shift+;`: コメントタブと索引タブの切替
- `Ctrl+;`: コメント欄または索引ツリーと盤面のフォーカス切替

## コメント

- 各局面ごとにコメントを保存できます。
- `記号等` をオフにすると、コメント欄の内容は消えずに非表示になります。
- タッチ主体の端末では、コメント欄は読み取り専用になります。

## 索引

- `索引` タブで現在局面に 3 階層のラベルを登録できます。
- 入力例は `2026`、`珠王戦`、`1` です。
- 同じ局面には 1 件だけ登録できます。
- ラベル名の重複登録はできません。
- 一覧の第 3 階層ラベルを選ぶと、その局面へ移動します。
- 非タッチ環境では、各階層を鉛筆ボタンで名前変更できます。
- `−` ボタンまたは `Delete` で削除できます。上位階層を消すと配下もまとめて削除されます。

### 索引のキーボード操作

| キー | 動作 |
|---|---|
| `↑` / `↓` | 項目移動 |
| `←` / `→` | 開閉、親子移動 |
| `Enter` | 第 3 階層ラベルを選択して局面へ移動 |
| `Delete` | 選択中の項目を削除 |

## 補足

- ページを閉じるときは確認ダイアログが表示されます。
- スマホやタブレット幅では、操作ボタンが画面下部に移動します。
- タッチ主体の端末では、索引の登録欄と名前変更ボタンは表示されません。
