# 連珠 検討用ボード

ブラウザで動作する連珠の棋譜検討ツールです。`index.html` を開くだけで使えます。

## 基本操作

- 盤面をクリックまたはタップすると石を置きます。
- 同じ局面から別の交点を選ぶと分岐を作れます。
- 既にある候補手を選ぶと、その手順に移動します。

## 手順の移動

| 操作 | キーボード | ボタン |
|---|---|---|
| 1手戻る | `←` / `j` | `<` |
| 1手進む | `→` / `k` | `>` |
| 前の分岐点へ | `↑` / `h` | `<<` |
| 次の分岐点へ | `↓` / `l` | `>>` |
| 現在手以降を削除 | `Delete` | — |

- 右クリックでも 1 手戻れます。
- `Ctrl+クリック` で交点にテキストを付けられます。

## ファイル

- `Ctrl+O` またはファイルアイコンの `読込` で JSON を開きます。
- `Ctrl+S` またはファイルアイコンの `保存` で保存します。
- JSON ファイルのドラッグ＆ドロップでも読込できます。
- iPad / iPhone などでは、ブラウザの挙動により保存はダウンロードになる場合があります。

## コメントと表示

- `Ctrl+Shift+;` で `コメント` と `索引` を切り替えます。
- `Ctrl+;` でコメントタブではコメント欄と盤面、索引タブでは索引ツリーと盤面のフォーカスを切り替えます。
- `Ctrl+F` または目アイコンの `コメント` でヒント表示を切り替えます。
- 目アイコンの `座標` で座標表示を切り替えます。

## 索引

- `索引` タブから現在局面に 3 階層のラベルを登録できます。
- 入力例は `例：2026`、`例：珠王戦`、`例：1` です。
- 同じ局面には 1 件だけ登録できます。ラベル名の重複もできません。
- 索引ツリーにフォーカスがある間は、枠がやや青くハイライトされます。
- 一覧の第3階層を選ぶと、その局面を盤面に表示します。
- 非タッチ環境では各階層を鉛筆ボタンで名前変更できます。
- 各階層を `−` ボタンで削除できます。上位階層を削除すると配下もまとめて削除されます。

### 索引のキーボード操作

| キー | 動作 |
|---|---|
| `↑` / `↓` | 項目移動 |
| `←` / `→` | 開閉、親子移動 |
| `Enter` | 第3階層ラベルを選択して盤面に表示 |
| `Delete` | 選択中の項目を削除 |

- 索引タブで `Ctrl+;` を押すと、盤面と索引ツリーのフォーカスを切り替えられます。
- タッチメイン端末では索引の登録欄と名前変更ボタンは表示されません。

## 補足

- ページを閉じる前に確認ダイアログが表示されます。
- スマホやタブレット幅では操作ボタンが画面下部に移動します。
